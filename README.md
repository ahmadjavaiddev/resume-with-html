# Personal Resume - HTML & CSS Assignment

Live Link: https://resume-with-html-rho.vercel.app/

This repository contains a single-page personal resume built using pure HTML and little bit CSS. The project replicates a specific design requirement, focusing on table structuring and semantic HTML.

## Project Overview

The goal of this project was to create a digital resume that accurately reflects a provided design mock-up. It includes sections for personal details, skills, experience, projects, and education.

### Key Features

- **Structure:** Semantic HTML tags (`header`, `section`, `footer`) for better accessibility and code readability.
- **Styling:** Custom CSS using internal styles to manage layout, typography, and spacing.
- **Table Design:** Added border using simple CSS properties.

---

## Setup & Installation

Since this project uses static HTML and little bit CSS.

1.  **Clone the repository:**

```
    git clone https://github.com/ahmadjavaiddev/resume-with-html
```

2.  **Navigate to the project folder:**

```
    cd resume-with-html
```

To view the resume:

1.  Locate the `index.html`.
2.  Double-click `index.html` to open it in your browser.
